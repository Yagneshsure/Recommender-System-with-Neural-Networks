##  Overview

Most recommender systems rely on static similarity metrics or matrix factorization, which fail to capture **non-linear and temporal relationships**.  
**NeuralRecoNet** leverages deep learning to model complex interactions, enabling more **dynamic, context-aware, and accurate recommendations**.

The framework is modular ‚Äî allowing experimentation with:
-  **Autoencoders** for latent feature learning  
-  **Recurrent Neural Networks (RNN / LSTM / GRU)** for sequential recommendation  
-  **Graph Neural Networks (GNN)** for relationship-aware suggestions
